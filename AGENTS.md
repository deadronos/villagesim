# AGENTS

Start with `docs/ARCHITECTURE/`.

Keep code changes aligned with those architecture docs. If your change updates the architecture itself, update an existing ADR or add a new one in `docs/ARCHITECTURE/gamedecisions/` or `docs/ARCHITECTURE/technicaldecisions/`.

Every handoff should leave one of these true: the codebase matches the docs, the docs match the codebase, or both.
